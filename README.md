# Enhanced Contra-Style Web Game / 增强版魂斗罗风格网页游戏

This is an enhanced version of the classic Contra-style side-scrolling shooter game created using HTML5 Canvas and JavaScript by 波特 AI Assistant.

这是由AI助手波特使用HTML5 Canvas和JavaScript创建的经典魂斗罗风格横版射击游戏的增强版。

### Visual Improvements / 视觉改进
- **Enhanced graphics**: Better visual effects with gradients, particle systems, and explosions
- **Animated backgrounds**: Dynamic starfield background with parallax effect
- **Visual feedback**: Particle effects for hits, explosions, and power-ups
- **Health bars**: Visual indicators for player and strong enemies
- **Improved UI**: Modern game interface with level, power-up indicators

- **增强图形**: 更好的视觉效果，包括渐变、粒子系统和爆炸效果
- **动画背景**: 动态星空背景和视差效果
- **视觉反馈**: 命中、爆炸和道具拾取的粒子效果
- **血量条**: 玩家和强大敌人的可视化指示器
- **改进界面**: 现代化游戏界面，包含等级和道具指示器

- **Multiple enemy types**: Different enemy behaviors (chasing, patrolling, normal)
- **Power-up system**: Rapid fire, multi-shot, and health restoration

- **Collectible items**: Pick up power-ups dropped by defeated enemies
- **Health system**: Player health management
- **Level progression**: Advance to harder levels as score increases

- **渐进难度**: 随着等级提升挑战增加

## How to Play / 游戏方法

### Controls / 控制方式
- Arrow keys (up, down, left, right) to move the character
- Spacebar to shoot
- P key to pause/resume the game

- 方向键（上、下、左、右）移动角色
- 空格键射击
- P键暂停/恢复游戏

### Game Elements / 游戏元素
- Green rectangle: Player character
- Yellow circles: Player bullets
- Red rectangles: Enemies
- Colored circles: Collectible power-ups
  - Red: Health restoration
  - Cyan: Rapid fire power-up
  - Magenta: Multi-shot power-up

- 绿色矩形: 玩家角色
- 黄色圆圈: 玩家子弹
- 红色矩形: 敌人
- 彩色圆圈: 可收集的道具
  - 红色: 生命恢复
  - 青色: 快速射击道具
  - 洋红色: 多发子弹道具
